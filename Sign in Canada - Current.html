Sign In Canada

Sign in to access Government of Canada services.

Use the sign-in option you registered with:

Not sure which sign-in to use?

You need to sign in with the same username and password every time you use My Account.

  • When you first registered for a My Canada Account, you selected a particular sign-in method.
  • You might have created a new Canada user id and password just for this account, and written it down or saved it in a Password manager.
  • Or when you registered, you may have decided to use your existing financial institution username and password rather than create a new set for Canada.
Want to switch to another authentication method?

Your application account is linked to the sign-in option you registered with.

  • You might have created a new Canada user id and password just for this account, and written it down or saved it in a Password manager.
  • Or when you registered, you may have decided to use your existing financial institution username and password rather than create a new set for Canada.
  • If you have multiple methods to sign in listed above to different government applications, you can consolidate them.
  • You can even register a previously unused option and switch known applications to it.
  • You will be shown list of applications known to be tied to your sign-in option. If an application is not listed but known to you, please sign in to that application before switching.

Note: If you have not previously logged in to an application through this portal, due to privacy that application account will not be migrated.
